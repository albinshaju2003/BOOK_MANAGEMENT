# 📚 Book Management System

A full-stack Book Management System built using **React.js (Frontend)**, **FastAPI (Backend)**, and **MongoDB (Database)**.

## 🚀 Features

- ✅ User Registration and Login (Authentication)
- 📖 Add New Books
- 👁️ View All Books
- ✏️ Edit Book Details
- ❌ Delete Book
- 🔗 Frontend-Backend API Communication
- 🛢 MongoDB Integration
- 💅 Clean UI with modular design
